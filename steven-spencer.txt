    1. What is Semantic HTML? 
		Semantic HTML is HTML that is used in a way to clearly separate different elements from each other.

    2. What is HTML used for? 
		HTML is used for creating websites and documents on the internet. (HyperText Markup Language)

    3. What is an attribute and where do we put it? 
		An attribute is a piece of code that provides additional information about HTML elements. Any HTML element can have an attribute, and they usually come in name/value pairs (name="value"). Classes, IDs, href, src.

    4. What is the h1 tag used for? How many times should I use it on a page?
		h1 tag is the main header. Should be used as few times as possible on the page, as it conveys the main/most important topic.

    5. Name two tags that have required attributes
		<a> (href=" "), and <img> (src=" ")

    6. What do we put in the head of our HTML document? 
		the head of our HTML document is a container for metadata. Typically defines the title, characters, styles, scripts, and other info.

    7. What is an id? 
		an id is an attribute that only applies to a single element

    8. What elements can I add an id to? 
		All html elements can have an id

    9. How many times can I use the same id on a page? 
		Once

    10. What is a class? 
		A class is an attribute that can apply to a more broad range of elements

    11. What elements can I add a class to? 
		All html elements can have a class

    12. How many times can I use the same class on a page? 
		As many times as you want

    13. How do I get my link to open in a new tab?
		<a href="www.google.com" target="_blank"</a>

    14. What is the alt attribute used for? 
		It is used for providing alternate information instead of a picture (for example) if the picture cannot be rendered 

    15. How do I reference an id?
		#id

    16. What is the difference between a section and a div
		A section is used to group a more broad range of code, whereas a div is more-specific

    17. What is CSS used for? 
		CSS is used for stylizing HTML elements

    18. How to we select an element? Example - every h2 on the page

    19. What is the difference between a class and an id? - Give me an example of when I might use each one
		class can be used multiple times throughout the HTML. If there's a website with multiple paragraphs that need the same style, a class would be used to style each one at the same time. An ID would be used for a single element, for example a picture, or single line of text, that only needs to be referenced once.

    20. How do we select classes in CSS?
		.class{ }

    21. How do we select a p element with a single class of “human””?
		p .human { }

    22. What is a parent child selector? When would this be useful? 
		Parent child selector will select the nested elements (for example, the <span> elements inside of a <div>). Would be useful for modifying specific parts of an element without having to write an unnecessarily long amount of extra code.

    5. In your own words, explain the box model. What is the fix for the box model? 
		The box model is the margin, border, and padding surrounding the content. It is used to create colored (or transparent) space around content to give it a little bit of "breathing room" and to better-stylize a website. The "fix" is box-sizing:border-box
